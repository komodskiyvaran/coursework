## **Разработка тестирующей программы для оценки знаний по языку программирования Java.**

- Техническое задание:
    1. При запуске программы должна появляться панель с кнопаками, где:
    - регистрация (связь с mySQL, JDBC)
    - авторизация
    - начать тестирование
    - просмотр результатов тестирования
    - выход из программы
    2. Необходимо сформировать тест в отдельном файле, должно быть 4 варианта ответа.
    3. Тести должен быть ограничен по времени (20 минут). После прохождения теста результат выводятся на экран и записываются в бд.
    4. При выборе кнопки “результаты тестирования” появляется список всех, кто прошел тестирование
       -    Фамилия - Имя - дата и время - процент правильных ответов
    5. Тест нельзя пройти повторно.   
    6. Необходимо использовать библиотеку JavaFX.
